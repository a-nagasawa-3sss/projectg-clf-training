# ① クラウドの基本概念

## クラウドの6つの特徴

| 項目            | 覚え方          |
| ------------- | ------------ |
| オンデマンドセルフサービス | 必要な時にすぐ利用    |
| 広帯域ネットワークアクセス | インターネット経由で利用 |
| リソースプーリング     | 利用者同士で共有     |
| 迅速な伸縮性        | すぐ増減できる      |
| 従量課金          | 使った分だけ支払う    |
| 測定可能なサービス     | 利用量を計測       |

# 最後の一言

CLF試験では細かな設定値よりも、**「このサービスは何をするものか」** を正しく判断できるかが重要です。

迷ったら、

* **Compute**
* **Storage**
* **Database**
* **Network**
* **Security**
* **Monitoring**
* **Analytics**
* **Cost**
* **Support**

のどのカテゴリに属するサービスかを考えると、正解にたどり着きやすくなります。
